📸 Crear la imagen PNG:

Para crear el archivo PNG, puedes:

  1. Ir a htmlcsstoimage.com
  2. Copiar el contenido de og-image.html
  3. Configurar: 1200x630 pixels
  4. Descargar como PNG
  5. Subir el PNG a la carpeta public como og-image.png
⚠️ IMPORTANTE: Una vez que subas og-image.png, la imagen debería aparecer automáticamente en WhatsApp.

🔧 Crear Favicon.ico (Para reemplazar icon de Vercel):

Para eliminar el icon de Vercel completamente:

  1. Ir a cloudconvert.com/svg-to-ico
  2. Subir public/favicon.svg
  3. Seleccionar tamaños: 16x16, 32x32, 48x48
  4. Descargar como favicon.ico
  5. Subir a la carpeta public
